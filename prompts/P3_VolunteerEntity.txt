Chapter: Persistence des données
Create Volunteer.java (extends BaseEntity).
Fields:

name, surname, email, phone, password (String)
rating (Double, default 0.0), ratingCount (int)
interests → List<String> (use @ElementCollection)
skills → List<String> (e.g., "First Aid", "Teaching")
availability → List<String> (e.g., "Monday", "Weekend")
Map to table: volunteers.
